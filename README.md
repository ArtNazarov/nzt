TypeScript testing library 
===

See sampleTests.ts

Usage
===
1) Compile `tsc sampleTests.ts`
2) Run `node sampleTests.js`

Author: Nazarov A.A. 2024

Installation
===
1) Install via NPM `npm install nzt-ts --save`
2) Import test engine `import NZT from 'nzt-ts'`
